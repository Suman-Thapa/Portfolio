Key Features

  • Data stored and managed in MySQL database
  • Built with Arduino for hardware integration

Smart Car Parking System

This Smart Car Parking System is built using Arduino, sensors, Python, and API integration to provide real-time parking management. Two sensors are used where one detects incoming cars and increases the car count, while the other detects outgoing cars and decreases the count. The system stores and updates data live in a MySQL database. When the parking reaches its maximum capacity, the gate automatically restricts entry to prevent overcrowding. It ensures efficient space management and real-time monitoring.
