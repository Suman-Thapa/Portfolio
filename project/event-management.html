Key Features

  • User authentication system(Login/Register)
  • Toast message Slider
  • Users can view available events and book tickets easily
  • After booking, user is redirected to invoice page
  • Users can view their booked events and invoices
  • Admin can add, edit, and delete events
  • Admin can view all bookings and invoices made by users

Event Management System

This Event Management System allows users to explore available events, select tickets, and complete bookings after a successful booking, users are redirected to an invoice page where they can view their booking details. The system also lets users track their booked events anytime. On the admin side, events can be fully managed by adding, editing, or deleting them, and all user bookings along with invoices can be viewed and controlled efficiently.
